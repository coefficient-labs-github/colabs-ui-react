```jsx
<span
  style={{
    display: 'grid',
    justifyItems: 'center',
    alignItems: 'center',
  }}
>
  <Text size={1}>Size 1</Text>
  <Text size={2}>Size 2</Text>
  <Text size={3}>Size 3</Text>
  <Text size={4}>Size 4</Text>
  <Text size={5}>Size 5</Text>
  <Text size={6}>Size 6</Text>
  <Text size={7}>Size 7</Text>
</span>
```

It also has different weights:

```jsx
<span
  style={{
    display: 'grid',
    gap: '1rem',
    justifyItems: 'center',
    alignItems: 'center',
  }}
>
  <Text weight="light">Size</Text>
  <Text weight="regular">Size</Text>
  <Text weight="medium">Size</Text>
  <Text weight="bold">Size</Text>
</span>
```
